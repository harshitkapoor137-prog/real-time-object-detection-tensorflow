# Real-Time Object Detection using TensorFlow SSD MobileNet

## Overview

This project implements a real-time object detection system using TensorFlow's SSD MobileNet model and OpenCV. It detects multiple objects in images by generating bounding boxes and confidence scores.

The project demonstrates practical applications of Computer Vision and Deep Learning using pre-trained TensorFlow models.

---

## Features

- Detects multiple objects in images
- Displays bounding boxes around detected objects
- Shows confidence scores
- Uses TensorFlow SSD MobileNet model
- Image preprocessing using OpenCV
- Easy to customize for different datasets

- TensorFlow
- OpenCV
- NumPy

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/real-time-object-detection-tensorflow.git
```

Move into the project

```bash
cd real-time-object-detection-tensorflow
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python object_detection.py
```
