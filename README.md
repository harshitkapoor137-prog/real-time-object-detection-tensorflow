# Real-Time Object Detection using TensorFlow

## Overview
This project detects objects in images using TensorFlow SSD MobileNet and OpenCV.

## Features
- Object Detection
- Bounding Boxes
- Confidence Scores
- Image Processing

## Technologies Used
- Python
- TensorFlow
- OpenCV
- NumPy

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run

python object_detection.py
